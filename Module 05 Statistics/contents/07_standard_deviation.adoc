= 표준 편차

산포도를 측정하는 표준 방법으로 중앙값을 근거로 한 IQR과는 달리, 표준 편차는 평균으로 부터 산포도를 측정합니다. 표준편차는 데이터가 평균에서 떨어져 있는 평균 거리라고 할 수 있습니다.
